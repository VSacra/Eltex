Комплиция происходит посредством Makefile. Для того, чтобы скомпилировать программу введите в консоль make program или же make run.
Для ручного теста необходимо скомпилировать программу и после вызвать её, сообщая аргументы. 
make test запустит программу с тестовыми аргументами program 123 45.67 test 1000 -5 3.14159
make test-empty запустит программу, не передевая ни одного аргумента, кроме названия исполняемого файла.
Чтобы удалить скомпилированный файл введите make clean
