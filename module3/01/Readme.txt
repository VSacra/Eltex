Комплиция происходит посредством Makefile. Для того, чтобы скомпилировать программу введите в консоль make program. Чтобы запустить для ручного теста - make run. 
make test запустит программу с тестовыми аргументами program 123 45.67 test 1000 -5 3.14159
make test-empty запустит программу, не передевая ни одного аргумента, кроме названия исполнительного файла.
Чтобы удалить скомпилированный файл введите make clean
